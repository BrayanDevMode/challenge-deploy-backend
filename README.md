# Full Stack Notes App - Ensolvers Challenge

Single Page Application (SPA) to manage notes, including features to create, edit, archive, delete, and categorize notes using tags.

## Technologies Used

### Backend
* **Java 21** (Compatible with Java 17+)
* **Spring Boot 3.x**
* **Spring Data JPA** (Persistence layer)
* **H2 Database** (File-based embedded database)

### Frontend
* **React 18**
* **Vite**
* **Bootstrap 5** (Styling and Responsive Design)
* **Axios** (HTTP Client)

## Prerequisites

Ensure you have the following installed on your system:
* **Java JDK 17** or higher (`java -version`)
* **Node.js** v18 or higher (`node -v`)
